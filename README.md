# Image Tools

一个轻量级的图片处理工具，支持图片预处理和批量合成，纯前端运行，无需后端服务。

## 📦 功能特性

### 1. 图片预处理
- 支持批量导入图片（拖拽或选择文件）
- 自动裁边（去除透明/白色边缘）
- 智能缩放至指定级别
- 统一画布尺寸输出
- 实时预览处理效果

### 2. 图片合成
- 支持多产品图与多背景图组合
- 自定义产品缩放比例
- 支持水印添加
- 多种导出格式（PNG/JPEG/WEBP）

### 3. 配置管理
- 默认参数本地保存
- 下次打开自动加载
- 无需安装任何软件

## 🚀 快速开始

### 直接使用

1. 打开 `image-tools.html` 文件
2. 在浏览器中使用（推荐 Chrome 或 Edge）
3. 无需网络，纯本地运行

## ⚙️ 配置说明

### 预处理参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 默认级别 | 图片缩放尺寸（5 档） | Level 3 (210px) |
| 画布尺寸 | 输出画布大小 | 1080×1080 |
| 裁边 | 去除透明/白色边缘 | 开启 |

### 合成参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 产品缩放比例 | 产品图放大/缩小 | 1.0 |
| 导出格式 | PNG/JPEG/WEBP | PNG |
| 压缩质量 | JPEG/WEBP 质量 | 95% |

## 💡 使用流程

### 预处理图片

1. 切换到「预处理工具」标签
2. 拖拽图片到上传区域，或点击「选择文件」
3. 调整级别和画布尺寸参数
4. 点击列表行预览效果
5. 点击「批量处理」开始处理
6. 点击「下载全部」保存结果

### 合成图片

1. 切换到「合成工具」标签
2. 上传产品图和背景图
3. 调整缩放比例和导出格式
4. 选择预览组合效果
5. 点击「批量合成」生成所有组合
6. 点击「下载全部」保存结果

## ❓ 常见问题

### Q: 如何保存配置？
A: 点击「配置」标签页，调整参数后点击「保存配置」，下次打开页面时会自动加载。

### Q: 图片保存在哪里？
A: 点击「下载全部」后，图片会下载到浏览器默认下载目录。
